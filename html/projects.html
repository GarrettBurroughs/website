Projects
Here's a list of the notable projects I've done
Alloy
An open source FIRST robot framework to aid teams in development. The framwork is designed in a way that is very easily customizeable, while abstracting away the tedious, boilerplate code that is often found in FRC and FTC robot code. The framework is also able to be implemented with almost all robot systems that have access to motors and sensors. Alloy comes built in with multiple different advanced control systems and integrations, but also allows the user to create their own utilities
Boost
An online platform that I developed during the 2018 All Star Code Summer Intensive. The goal of the project was to make a new media platform that focuses on new content creators, and short 30, second clips for them to show themselves off to the world.
Bloxris
A "Tetris" like game developed during one of my computer science classes at my highschool. This implementation of tetris is written in the java processing enviornment and contains all basic gameplay elements
Mania
A "Step Mania" type game written in processing. This is my own implementation of a computer based dance, dance revolution. The project also includes a converter from the popular "osu!mania"
osuBot
A discord bot written in node.js to help with the management of osu torunaments. By using discord.js and the osu!API, this project is able to capture match information from commands run within a discord server, and then log the match results both to the server, and an external database, this case being a simple google spreadsheet.
Genetic Algorithm
A genetic algorithm implementation written in pure java. This project contains the main genetic algorithm implementation, and allows the user to create their own versions by defining the actions of a specific organism.
